독후활동 - 1:1맞춤형 첨삭 지도

[학생]

독후활동 첨삭 지도는 P2P방식으로 학생과 선생님이 연결됩니다. 학생은 충전 후 문제 풀고 첨삭요청할 수 있고 독서 포트폴리오(내 서재)를 만들 수 있습니다. 장르별, 초등, 중등, 고등으로 구분되어 있습니다.

1. 메인 (링크)
- 초등학생, 중학생, 고등학생으로 구분
- 포트폴리오 페이지

2. 리스트 (링크)
- 충전 후 구매

3. 디테일 (링크1) (링크2) (링크3)
- 책 상세내용과 문제 기본 정보 안내
- 결제 후 문제 풀수 있음.

4. 문제풀기 (문제 풀기 링크) (문제 풀기 완료 링크)
- 주제별 문항 수 다름
- 임시저장 가능
- 문제풀이 완료 후 첨삭요청

5. 첨삭 요청 / 첨삭 완료 (링크)
- 선생님 선택 가능
- 첨삭요청 시간 조율 가능 (긴급 / 일반)

6. 포트폴리오 (링크)
- 풀었는 문제, 첨삭완료된 문제 등 데이터 기록.
- 풀었는 문제는 인쇄가능

7. 기타기능
- 결제기능(충전 후 차감형태)

[선생님]

가입 시 관리자의 승인을 받아야 선생님으로 등록할 수 있습니다. 등록된 선생님은 첨삭요청을 받기 위해 '첨삭 가능한 도서'를 등록 해야 요청서를 받을 수 있습니다. 문제등록은 관리자의 승인을 받아야 하며, 승인을 받았더라도 문제 제출 완료 시 문제검토를 관리자에게 받아야 합니다.
캐시는 첨삭했을 경우, 문제등록 했을 경우에 선생님께 지급되며 일정 금액이 되면 환급되는 방식.
구체적 캐시금액은 추 후 논의.

1. 어드민 페이지로 접속
- 학생페이지로 전환 가능

2. 주요 루트 : 어드민 - 첨삭요청 - 첨삭 - 완료 시 첨삭내역에 데이터 쌓임

3. 메뉴구분
0. 계정정보 1. 대시보드 (기본적인 통계자료)
2. 캐시관리 (링크)
3. 첨삭요청 (핵심) (리스트 링크) (첨삭 페이지 링크)
4. 첨삭내역 (링크)
5. 문제은행
6. 문제등록 (책 등록 링크) (문제 등록 링크)

4. 첨삭요청 알림
- 신규 요청이 들어올 경우 이메일로 알림, 긴급 / 일반 구분됨

[전체 관리자]

1. 학생, 선생님으로 구분해서 관리 가능해야함
2. 문제관리
3. 첨삭관리
4. 선생님 승인 권한
5. 캐시관리
6. 기타 모든 통계치 등
